Contributing to AYSON

Thank you for your interest in contributing to AYSON.

We appreciate bug reports, suggestions, documentation improvements, and code contributions.

---

Reporting Bugs

Before opening a new issue:

- Make sure the problem hasn't already been reported.
- Include your Android version and device model.
- Describe the steps required to reproduce the issue.
- Include screenshots if applicable.

---

Feature Requests

Feature suggestions are welcome.

Please explain:

- What you would like to add.
- Why it would be useful.
- Any alternative solutions you have considered.

---

Pull Requests

Before submitting a pull request:

- Keep changes focused on a single feature or fix.
- Follow the existing coding style.
- Test your changes before submitting.
- Write clear commit messages.

---

Code of Conduct

Please be respectful and constructive when participating in discussions.

---

Thank you for helping improve AYSon.
