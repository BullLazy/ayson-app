Security Policy

Supported Versions

Security updates are currently provided for the latest public release only.

Version| Supported
Latest Release| ✅
Older Releases| ❌

---

Reporting a Vulnerability

If you discover a security vulnerability in AYSon, please do not create a public GitHub Issue.

Instead, report it privately by contacting the maintainers.

Please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- The affected version
- Any proof of concept (if available)

We will investigate all legitimate reports and work to resolve confirmed vulnerabilities as quickly as possible.

---

Responsible Disclosure

Please allow us reasonable time to investigate and fix the issue before publicly disclosing any vulnerability.

Thank you for helping keep AYSon secure.
